# REAPER v7.0 Release Notes

These release notes cover major changes introduced in **REAPER version 7.0**.

---

## Who Should Read This

This guide is intended for:

- Users upgrading from REAPER 6.x
- Intermediate producers and musicians familiar with DAWs (Digital Audio Workstations)
- Anyone looking for a concise, structured breakdown of what’s new in v7.0

> If you're unfamiliar with the term **DAW**, see our upcoming documentation: _[What is a DAW? →](../../conceptual-docs/what-is-a-daw.md)_

---

## System Compatibility

REAPER 7.0 is compatible with:

| OS         | Requirements                                                |
|------------|-------------------------------------------------------------|
| **Windows**| Windows 7 or newer, 32/64-bit systems                        |
| **macOS**  | macOS 10.5 or later, including support for Apple Silicon    |

Existing users can install it over REAPER 6.x without losing any configurations.

---

## Highlights at a Glance

- New default theme with updated interface and UX design
- Fully redesigned **track lane system**
- Smarter automation with lane-specific features
- Significant improvements to routing visibility and behavior
- Better MIDI editor layout and reduced latency

---

## Major Features in Version 7.0

---

### New Default Theme

The entire REAPER interface has been redesigned in version 7.0 with the goal of improving visibility, spacing, and overall usability.

#### What Changed:

- **Floating track control panels** for more flexible window layouts  
- **More whitespace and visual grouping** to reduce eye fatigue  
- **Updated color palette** and contrast settings  
- Support for **dynamic resizing** based on zoom level

#### Why It Matters:

Users switching from REAPER 6 will immediately notice a cleaner, more navigable UI. The redesign does not affect functionality, but significantly enhances workflow comfort—especially in large sessions.

---

### Advanced Track Lane Management

One of the most powerful features introduced in 7.0 is a **revamped track lane system**, designed to simplify editing workflows.

#### Key Improvements:

- **Automatic lane creation** during recording and comping  
- **Dedicated editing per lane** (mute, solo, FX support)  
- Ability to **collapse, reorder, or hide lanes** for better session management

#### Use Case Example:

Instead of stacking takes into layers, each take now sits in a **clearly labeled lane**, making it easier to comp vocals or guitar tracks. You can switch between takes without navigating menus—just click the lane header.

---

### Enhanced Routing Interface

Routing in REAPER is known for its flexibility—and now, in version 7.0, it's also **visual**.

#### What's New:

- Redesigned **Routing Matrix** with better label clarity  
- New **visual cues for signal direction**, preventing misrouting  
- **Drag-and-drop routing** between tracks via the matrix

#### Why It Matters:

For users managing complex sessions with buses and submixes, the new matrix reduces errors by making signal paths easier to understand at a glance.

---

### Improved MIDI Workflow

REAPER’s MIDI editor in v7.0 gets both visual and functional upgrades.

#### Updates Include:

- **Smarter grid behavior** based on note context  
- Clearer **note velocity coloring** and contrast  
- Improved **piano roll layout** and scrolling performance  
- Reduced **MIDI latency** on playback and recording

#### Who Benefits:

Electronic musicians, composers, and sound designers using REAPER for heavy MIDI work will now experience better responsiveness and visibility—especially during fast-paced editing.

---

### Additional UX Updates

Some smaller (but impactful) improvements include:

| Feature                          | Benefit                                  |
|----------------------------------|------------------------------------------|
| Improved undo behavior           | More consistent with multi-step edits    |
| Track icon repositioning         | Icons now dock logically next to names   |
| Tooltip timing and design        | Faster, more readable contextual info    |
| Cross-platform file consistency  | Fewer issues when switching OS or drives |

## Known Limitations

- Some users may prefer the REAPER 6 theme. You can switch via:  
  `Options → Themes → Default_6.0`
- Certain older plugins may need visual scaling adjustments

## Upgrade Instructions

1. Back up your REAPER configuration (optional but recommended)  
2. Download the 7.0 installer from: [https://www.reaper.fm/download.php](https://www.reaper.fm/download.php)  
3. Install over your existing REAPER install  
4. Launch and verify previous sessions load correctly
