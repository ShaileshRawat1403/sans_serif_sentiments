<a name="version-2.1.0-anchor"></a>
### Version 2.1.0 (June 28, 2025)

This release focuses on enhancing the document review process, improving notifications, and adding new sharing capabilities.

<a name="new-features-anchor"></a>
#### New Features

* **Custom Reviewer Chains:** Users can now define custom, sequential reviewer chains for each document, allowing for more flexible and tailored approval workflows.
    * *Benefit:* Adapt review processes precisely to your organizational needs, streamlining complex approvals.
* **Public Verification Portal:** Documents sealed on the blockchain can now be shared via a unique public link, allowing external parties (e.g., auditors, partners) to verify integrity without logging into ImmutableInk.
    * *Benefit:* Increase transparency and trust with external stakeholders by providing direct, verifiable proof of document integrity.
* **Version Compare Tool:** Introducing a new visual tool that highlights differences between any two sealed versions of a document.
    * *Benefit:* Easily track changes and understand the evolution of your documents over time.

<a name="improvements-anchor"></a>
#### Improvements

* **Enhanced Notification System:** Receive more granular notifications for document status changes, review requests, and comments. You can now customize notification preferences in your User Settings.
    * *Benefit:* Stay informed about your documents and team activity more efficiently.
* **Faster Document Loading:** Optimized document rendering for large files, significantly reducing loading times in the editor and viewer.
    * *Benefit:* Experience a smoother and more responsive editing and viewing experience.
* **Improved Search Functionality:** Search now includes fuzzy matching and better relevance ranking, making it easier to find documents even with partial keywords.
    * *Benefit:* Locate your documents quickly and accurately.

<a name="bug-fixes-anchor"></a>
#### Bug Fixes

* Fixed an issue where some users occasionally experienced duplicate review requests.
* Resolved a display bug affecting table formatting in the document editor on certain browser versions.
* Corrected an issue preventing successful document sealing when specific special characters were present in the document title.

<a name="known-issues-anchor"></a>
#### Known Issues

* **Printing Public Verification Portal:** When printing pages from the Public Verification Portal, some elements might not render perfectly in certain browser print previews. (Workaround: Use browser's "Save as PDF" option instead of direct print for a better layout.)

<a name="version-2.0.0-anchor"></a>
### Version 2.0.0 (May 15, 2025)

This was a major release introducing the core blockchain sealing functionality and initial user collaboration features.

<a name="new-features-2.0.0-anchor"></a>
#### New Features

* **Blockchain Sealing:** Introduced the core capability to cryptographically seal documents on an immutable ledger, providing tamper-evident records.
* **Digital Signatures:** Enabled users to digitally sign documents during the sealing process.
* **Basic Review Workflow:** Implemented the initial functionality for assigning reviewers and capturing their approvals.

<a name="improvements-2.0.0-anchor"></a>
#### Improvements

* Initial user interface redesign for improved navigation.
* Enhanced document upload stability for common file formats.

<a name="bug-fixes-2.0.0-anchor"></a>
#### Bug Fixes

* Addressed minor display issues on mobile devices.
* Fixed an issue with password reset emails occasionally being delayed.
