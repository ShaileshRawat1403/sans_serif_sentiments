# 📚 GitBook Glossary

Use this glossary to understand key GitBook terms, roles, and features.  
Each entry includes a clear definition and links to related documentation where applicable.

---

| Term | Definition | Related Guide |
|------|------------|----------------|
| **Access Control** | Defines who can view, edit, or administer a space or workspace | [Setup Guide](./gitbook-setup-guide.md) |
| **Admin (Role)** | User role with full access to settings, members, and space visibility | [Setup Guide](./gitbook-setup-guide.md) |
| **Block Mode** | Visual editing mode using content blocks (text, headings, images, embeds) | [Editor Guide](./gitbook-editor-guide.md) |
| **Draft** | Unpublished content visible only to workspace members | [Getting Started Guide](./gitbook-getting-started.md) |
| **Editor (Role)** | Can create, modify, and publish content | [Setup Guide](./gitbook-setup-guide.md) |
| **Git Sync** | Optional feature to sync content from a GitHub repository | [Overview](./gitbook-overview.md) |
| **Markdown Mode** | Editing mode using Markdown syntax (`**bold**`, `# Heading`) | [Writing Modes Reference](./gitbook-writing-modes-reference.md) |
| **Page** | A single content document within a GitBook space | [Getting Started Guide](./gitbook-getting-started.md) |
| **Publish** | Make a draft page or space publicly visible | [Getting Started Guide](./gitbook-getting-started.md) |
| **Role** | Determines a member’s permissions (Viewer, Editor, Admin) | [Setup Guide](./gitbook-setup-guide.md) |
| **Slash Command (`/`)** | Command to insert blocks quickly in the editor | [Editor Guide](./gitbook-editor-guide.md) |
| **Space** | A container for related documentation pages | [Overview](./gitbook-overview.md) |
| **Viewer (Role)** | Read-only user who cannot create or modify content | [Setup Guide](./gitbook-setup-guide.md) |
| **Workspace** | Your top-level environment in GitBook, where you manage spaces and members | [Overview](./gitbook-overview.md) |

## 🔎 How to Use

- Use **Ctrl/Cmd + F** to search terms  
- Click any link to explore the full guide  
- Visit the [Editor Guide](./gitbook-editor-guide.md) for examples of block usage and formatting
