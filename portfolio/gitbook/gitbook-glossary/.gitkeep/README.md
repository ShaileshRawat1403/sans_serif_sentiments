# 📚 GitBook Glossary

Use this glossary to understand key GitBook terms, roles, and features.  
Each entry includes a clear definition and links to related documentation where applicable.

---

| Term | Definition | Related Guide |
|------|------------|----------------|
| **Access Control** | Defines who can view, edit, or administer a space or workspace | [Setup Guide](./gitbook-setup-guide.md) |
| **Admin (Role)** | Full-access user who can manage settings, users, and spaces | [Setup Guide](./gitbook-setup-guide.md) |
| **Block Mode** | Visual writing mode using blocks like headings, images, and embeds | [Editor Guide](./gitbook-editor-guide.md) |
| **Draft** | Unpublished page or space, visible only to team members | – |
| **Editor (Role)** | Can write and publish content but cannot manage workspace settings | [Setup Guide](./gitbook-setup-guide.md) |
| **Git Sync** | Sync a GitHub repo with a GitBook space for version-controlled content | – |
| **Markdown Mode** | Plain text writing mode using Markdown syntax | [Writing Modes Reference](./gitbook-writing-modes-reference.md) |
| **Page** | A single document within a GitBook space | [Getting Started](./gitbook-getting-started.md) |
| **Publish** | Make a page or space visible to external users | [Getting Started](./gitbook-getting-started.md) |
| **Role** | Defines a member’s permissions (Viewer, Editor, Admin) | [Setup Guide](./gitbook-setup-guide.md) |
| **Slash Command (`/`)** | Type `/` to access block options (e.g., heading, quote, image) | [Editor Guide](./gitbook-editor-guide.md) |
| **Space** | A container for related pages and sections within a workspace | [Overview](./gitbook-overview.md) |
| **Viewer (Role)** | Read-only access to documentation within private spaces | [Setup Guide](./gitbook-setup-guide.md) |
| **Workspace** | Your overall environment in GitBook, containing spaces and users | [Overview](./gitbook-overview.md) |

## 🔎 How to Use

- Click any link to learn more
