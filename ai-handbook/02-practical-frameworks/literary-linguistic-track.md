# Literary & Linguistic Track

_Part of the Prompt Dissection Lab_  
A track designed for writers, deep thinkers, and those crafting nuance.

---

## Why This Track Exists

Most guides focus on clarity — and that’s good.

But what if your goal isn’t clarity?

What if you’re trying to:

- Stir emotion  
- Challenge norms  
- Echo a voice  
- Shape a moment  

This track is for **those who write to move people** — not just inform them.  
It uses tools of literature, rhetoric, and composition to craft prompts that feel more **artful** and **intentional**.

---

## How AI Views Information

AI doesn’t “think” — it completes.

But **what it completes** is based on how it reads your instructions across 5 internal layers:

| Layer      | Description                                   |
| ---------- | --------------------------------------------- |
| Format     | Table? Paragraph? Poem? Dialogue?             |
| Style      | Playful, formal, cynical, empathetic          |
| Tone       | Optimistic? Detached? Urgent?                 |
| Guide      | Role/voice to follow (e.g., teacher, analyst) |
| Nuance     | Emotion, subtle meaning, subtext, metaphor    |

Knowing this, you can **reverse-engineer these levers** into your prompt.

> _Don’t just write what you want — shape how you want it to feel._

---

## Why Linguistics and Literary Devices Work

Because **AI was trained on language, not logic**.

Large Language Models (LLMs) like GPT or Claude were trained on:

- Novels, essays, scripts, Wikipedia, web content  
- Dialogues, debates, poetic forms, rhetorical arguments  

That means AI is excellent at:

- Recognizing literary and stylistic patterns  
- Emulating rhetorical voice  
- Switching narrative styles  

So if you learn to speak in **rhetorical, metaphorical, or artistic patterns** — the AI will follow.

> _Prompting is not coding. It’s closer to poetry than programming._

## Rhetorical Device Prompts

These help you _guide_ the AI’s emotional or persuasive structure.

| Goal           | Prompt Example                                                                                |
| -------------- | --------------------------------------------------------------------------------------------- |
| Persuasion     | “Use **ethos** to build trust, then **logos** to justify, and **pathos** to end emotionally.” |
| Urgency        | “Write using **kairos** — show why this matters _right now_.”                                 |
| Audience-Aware | “Write a rebuttal using **logos**, but anticipate the emotional objection using **pathos**.”  |
| Reframe        | “Take this emotional story and **flip the rhetoric** — make it sound logical and cold.”       |

---

## Thought Scaffolding Models

These prompt styles encourage **cognitive reasoning** — not just verbal completion.

| Model                 | Example Prompt                                                                                     |
| --------------------- | -------------------------------------------------------------------------------------------------- |
| Socratic              | “Challenge this assumption step by step. Start with a question, not a statement.”                  |
| Dialectic             | “Present two opposing views on this topic, then offer a synthesis.”                                |
| Narrative Framing     | “Explain this concept as a journey — with conflict, turning point, and resolution.”                |
| Moral Reasoning       | “Explore the ethical side using a dilemma. Present both outcomes and ask: ‘Which cost is higher?’” |

Tip: This is _incredibly useful_ for essays, critical writing, or human-AI dialogues in philosophy, policy, or psychology.

## Stylistic Mode Switches

AI can shape-shift. But only if you tell it **what mood, aesthetic, or tradition** to mimic.

| Mode           | Prompt Example                                                     |
| -------------- | ------------------------------------------------------------------ |
| Satire         | “Rewrite this as a dark satire, like Orwell meets Black Mirror.”   |
| Minimalist     | “Write this using short, sharp lines. Channel Hemingway.”          |
| Maximalist     | “Make this lush and sensory, like Nabokov or Wolfe.”               |
| Gothic         | “Give this a shadowy, brooding tone — like Poe with tech.”         |
| Surreal        | “Imagine Dali painted this idea. Now describe it in text.”         |
| Academic       | “Turn this explanation into a dry academic abstract in APA style.” |

Try combining:  
“Summarize the article using **logos** and **kairos**, in a **satirical tone**.”

---

## Bonus: Narrative Tension & Pacing Tricks

Want to **guide rhythm and reveal**? Try:

| Technique    | Prompt                                                                  |
| ------------ | ----------------------------------------------------------------------- |
| Slow Reveal  | “Tell this story in 3 paragraphs. In each, reveal one emotional layer.” |
| Contrast     | “Begin with optimism. End with subtle dread.”                           |
| Mirror       | “Make the beginning and ending echo — use parallel words.”              |
| Shift Midway | “Halfway, change the voice from certain to doubtful.”                   |

Use these to create prompts that **live on the edge of emotion and structure**.
