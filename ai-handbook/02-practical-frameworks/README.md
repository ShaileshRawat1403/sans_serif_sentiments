# Practical Prompting Frameworks

_A modular lab for thinking and writing with AI — not just typing at it._

This section hosts **structured prompting strategies** that combine clarity, intent, rhetorical flow, and usability — built for writers, designers, educators, and thinkers who want to collaborate better with language models.

## ✅ When to Use These

Use these frameworks when you want to:

- Prompt with more **control, flow, and reasoning**
- Align AI output with **readability, tone, or user goals**
- Explore multiple modes of asking — from **clear directives** to **literary expression**

These are reusable tools — adapt them to your writing or design environment.
