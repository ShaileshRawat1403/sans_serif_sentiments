# 01  foundations
## Who This Is For & How to Use It

### 🎯 Intended Audience
This handbook is for people who:
- Feel confused by AI’s confidence  
- Are frustrated by vague answers  
- Have been left behind in a world moving too fast  
- …

### 🛑 What You Don’t Need
- No developer background  
- No deep AI-internals knowledge  
- No memorized prompt templates  

> **You do need:** curiosity, patience, and the humility to rethink habits.

### 📖 What You’ll Learn
- Reflect on your AI interactions  
- Craft meaningful prompts, not just functional ones  
- Understand how invisible beliefs shape your questions  
- …

### 🧭 How to Use This Guide
| Reader Type               | Approach                                         |
| ------------------------- | ------------------------------------------------ |
| **Curious Beginner**      | Go slow—follow reflection prompts.               |
| **Overwhelmed User**      | Skim first, then re-read examples in depth.      |
| **Intentional Thinker**   | Engage with philosophical questions.             |
| **Strategic Educator**    | Adapt sections for teaching others.              |

#### A Final Ritual
Before moving on, ask yourself:
1. Did I pause to truly reflect?  
2. What assumption did I uncover?  
3. What will I try differently next time?
