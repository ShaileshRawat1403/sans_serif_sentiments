# Who This Is For & How to Use It

### 🎯 Intended Audience
This module is for people who:
- Want to get better results from AI  
- Struggle to translate intent into questions  
- Feel they’re just “copy-pasting” templates  
- …

### 🛑 What You Don’t Need
- No developer background  
- No formal linguistics training  
- No pre-memo­rized prompt libraries  

> **You do need:** curiosity, experimentation, and a willingness to iterate.

### 📖 What You’ll Learn
- How to break down your intent into modular prompts  
- Tricks for priming AI with context and constraints  
- When to chain prompts versus refine one  
- …

### 🧭 How to Use This Guide
| Reader Type               | Approach                                         |
| ------------------------- | ------------------------------------------------ |
| **Prompt Newcomer**       | Follow step-by-step examples, then tweak.        |
| **Power User**            | Compare “before/after” prompt variations.       |
| **Team Lead**             | Adapt patterns into your org’s prompt playbook. |
| **Educator**              | Use exercises to teach prompt strategy.         |

#### A Final Ritual
Before moving on, ask yourself:
1. Which prompt “hack” will I try first?  
2. How did the model’s output change?  
3. What hypothesis will I test next?
