# sans-serif-sentiments

*A writing portfolio where clarity commits, and fluff gets forked.*

This is not just a repository.  
It’s a rebellion against persuasive pretension, bloated frameworks, and jargon-scented PDFs.  
Here, we write to explain, not to impress.  
We write to teach, not to manipulate.  
We write to version our thoughts—truthfully.

Welcome to a documentation playground that respects intelligence, time, and intent.

---

## from kit-kit to git-git

Marketing bored me with its usual *kit-kit*.  
So, as a writer, I caught the nearest rhyme—and found **Git to commit**.

What started as a pun turned into a revelation.

Under the rock called “Marketing” lies a world of Git—a place where writing isn’t dressed up, it’s tracked.  
Where edits aren’t hidden in redlines but logged in real time.  
Where you don’t pitch, you version.  
And you don’t persuade—you prove.

Here’s how the shift feels:

|      | Marketing                             | Git                                                |
|------|----------------------------------------|-----------------------------------------------------|
| **Truth** | Bend it to fit a story                 | Brutally honest (tracks every mistake)              |
| **Versioning** | FINAL_final_v2_REAL.pptx           | `git commit -m "actual change"`                     |
| **Audience** | Persuade                             | Inform                                               |
| **Mistakes** | Smooth over                          | Preserve and reflect                                 |
| **Identity** | Copywriter, Content Designer, etc.   | Technical Writer, Information Developer              |
| **Clarity** | Optional                              | Mandatory                                             |
| **Results** | Vibes-based                           | Version-controlled impact                            |
